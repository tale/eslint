**Aarnav Tale's personal ESLint Configuration.**<br>
- React & JSX/TSX
- Node (CommonJS & ESM)
- Sensible Defaults (ES6)

## Usage
Edit your `.eslintrc` configuration to include the following
```json
{
	"extends": "tale"
}
```
